A comprehensive e-commerce platform that allows users to browse products, add items to their cart, and place orders. The app is designed to provide a smooth and interactive shopping experience.

## Features
- **Product Listing**: Displays a dynamic list of products with images, prices, and descriptions.
- **Search and Filter**: Users can search and filter products by category, price, and rating.
- **Shopping Cart**: Add, remove, and update quantities of items in the cart.
- **Order Management**: Place orders and view order history.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (for storing product data, cart items, and order history)
- **APIs**: Razorpay API for payment integration
